# Nosetalgia

«Nosetalgia» – парфюмерный бренд, изготавливающий по индивидуальному заказу ароматы, связанные с воспоминаниями.

Интерактивный веб-плакат представляет собой процесс создания индивидуального аромата, где можно создать собственную парфюмерную композицию, выбрать упаковку и протестировать полученный парфюм.
